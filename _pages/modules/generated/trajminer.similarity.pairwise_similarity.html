trajminer.similarity.pairwise_similarity¶

trajminer.similarity.pairwise_similarity(X, Y=None, measure=None, n_jobs=1)¶

Compute the similarity between trajectories in X and Y.

X : array-like, shape: (n_trajectories_X, n_points, n_features)
Input data.
Y : array-like, shape: (n_trajectories_Y, n_points, n_features)
Input data. If None, the output will be the pairwise similarities between all samples in X.
measure : SimilarityMeasure object (default=None)
The similarity measure to use for computing similarities. See trajminer.similarity.
n_jobs : int (default=1)
The number of parallel jobs.
similarities : array
An array with shape (n_trajectories_X, n_trajectories_Y).
